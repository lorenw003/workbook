interactive media

loren wilson

Week 1: This was our first week and introduction to interactive media and WWW. We considered their meaning through how we interact with them - gestures - and their structure or anatomy. We set up VS code and got started exploring html.

Some things that inspired me…Mosiac Virus, Myriad, Glitch Textiles, Maria Szakats

Things I learned…how to set up a html page, basic html structure

Week 2: This week we considered how the web has evolved overtime resulting in many different design styles and layouts. We began our own designs starting ideation and learning how to design for the web physically - prototypes - and digitally.

Some things that inspired me…Spellverse, Movie Palace Online, 45 Denim Kurata

Things I learned…paper and modular prototyping, crazy eights, githhub

Week 3: This week focused on learning how to publish to the WWW specifically considering grid (layout), responsive design and typography. These considerations informed the personalisation of our workbooks including learning how to use new html tags and css. As well as personal research for our unique visions.

Some things that inspired me…Parametro studios, Antione Wodniack, Agorespace

Things I learned…responsive/fluid design, media queries, variable fonts, custom fonts, audio, video
